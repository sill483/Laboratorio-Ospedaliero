# Laboratorio-Ospedaliero
Un applicazione che gestisce l'occupazione del laboratorio. Le sue funzionalità sono: 
- L'inserimento di un nuovo paziente con laboratorio e dottore di  riferimento
- Controllo se disponibile il laboratorio
- Visualizzazione di tutti i laboratori occupati in quel giorno
- Ricerca tramite motore
- Eliminazione delle vecchie occupazioni dei laboratori.
